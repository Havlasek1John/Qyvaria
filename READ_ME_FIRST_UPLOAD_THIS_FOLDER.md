# Upload this folder to GitHub

This folder contains only root-level files, with no hidden `.github` folder and no subfolders.

Use this version if GitHub web upload gives an error.

## Upload

1. Unzip this ZIP.
2. Open `https://github.com/Havlasek1John/Qyvaria`.
3. Select the default branch `Qyvaria`.
4. Click **Add file → Upload files**.
5. Select all files inside this folder.
6. Commit.

Do not upload the ZIP file itself.
