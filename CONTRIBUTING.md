# Contributing to Qyvaria

Thank you for contributing to Qyvaria.

Unless you explicitly mark a submission as **Not a Contribution**, any material
intentionally submitted for inclusion in this project is offered under the
Apache License 2.0, consistent with Section 5 of that licence.

## Contribution rules
- Submit only work you own or are authorized to contribute.
- Do not submit secrets, credentials, private keys, or confidential data.
- Preserve licence and NOTICE information.
- Document third-party additions.
- Add or update tests when practical.
