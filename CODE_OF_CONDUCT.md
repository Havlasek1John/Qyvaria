# Code of Conduct

## Our standard

This project aims to be respectful, constructive, and useful.

Participants should:

- Use welcoming and inclusive language.
- Respect different viewpoints and experiences.
- Give constructive feedback.
- Focus on what is best for the project and its users.

## Unacceptable behavior

Unacceptable behavior includes harassment, threats, hate speech, public posting of private information, and deliberate disruption.

## Enforcement

The maintainer may remove comments, close issues, or restrict participation when needed to protect the project.
