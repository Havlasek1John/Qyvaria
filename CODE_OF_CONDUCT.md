# Code of Conduct

Qyvaria collaboration should be respectful, constructive, and focused on the project.

## Expected
- respectful technical discussion;
- good-faith collaboration;
- clear issue reports;
- careful handling of security concerns.

## Not acceptable
- harassment;
- hate speech;
- doxxing;
- threats;
- malicious code submissions;
- deliberate disruption.
